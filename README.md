# JellyfinMobile

A minimal React Native (Expo) client for Jellyfin.

## Features

- Add and connect to any Jellyfin server (HTTP/HTTPS, IP or hostname, VPN-friendly)
- User authentication with session persistence
- Dashboard with continue watching and library browsing
- Library view with poster grid
- Settings with server info, user profile, and session management

## Usage

```bash
npm install
npx expo start
```

Scan the QR code with Expo Go, or press `a` for Android / `i` for iOS.

## Tech

React Native, Expo SDK 54, expo-router, zustand, axios, FlashList.
